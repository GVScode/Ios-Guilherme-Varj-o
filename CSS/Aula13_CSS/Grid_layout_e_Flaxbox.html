Violão Folk

O que é um violão folk? Resultado de imagem para violão folk Violão Folk: Possuem uma caixa maior, o que deixa o timbre mais encorpado, ideal para sons mais graves. O braço desse modelo é um pouco mais fino e nele são utilizadas cordas de aço, que são mais tensas e podem dificultar um pouco o desenvolvimento da mão esquerda (caso pratique como destro).
